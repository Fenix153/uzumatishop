# 🤖 Установка Discord бота — Пати Узумати

## Что нужно сделать по шагам

---

## ШАГ 1 — Создать бота в Discord

1. Открой https://discord.com/developers/applications
2. Нажми **"New Application"** → введи имя (например "Пати Узумати")
3. Слева нажми **"Bot"** → **"Add Bot"** → подтверди
4. Нажми **"Reset Token"** → скопируй токен — вставишь в `bot.py`
5. Ниже включи все три переключателя:
   - ✅ PRESENCE INTENT
   - ✅ SERVER MEMBERS INTENT
   - ✅ MESSAGE CONTENT INTENT
6. Слева нажми **"OAuth2"** → **"URL Generator"**
7. Отметь: `bot` + `applications.commands`
8. В Bot Permissions отметь:
   - ✅ Manage Roles
   - ✅ Send Messages
   - ✅ Read Messages/View Channels
9. Скопируй ссылку внизу → открой в браузере → добавь бота на свой сервер

---

## ШАГ 2 — Узнать нужные ID

**Включи режим разработчика в Discord:**
Настройки → Расширенные → Режим разработчика ✅

- **ID сервера** — правой кнопкой на название сервера → "Копировать ID"
- **Твой ID** — правой кнопкой на своё имя → "Копировать ID"
- **ID канала для логов** — правой кнопкой на нужный канал → "Копировать ID"
- **ID ролей** — Настройки сервера → Роли → правой кнопкой на роль → "Копировать ID"

---

## ШАГ 3 — Настроить bot.py

Открой файл `bot.py` и заполни в самом верху:

```python
BOT_TOKEN      = "твой_токен_сюда"
GUILD_ID       = 123456789  # ID сервера
ADMIN_ID       = 123456789  # Твой ID
LOG_CHANNEL_ID = 123456789  # ID канала для уведомлений

ROLE_MAP = {
    "VIP Роль":    123456789,   # ID роли VIP
    "Буст x1":     123456789,   # ID роли Буст
    # и т.д. — название должно совпадать с названием товара на сайте
}
```

---

## ШАГ 4 — Установить Python и запустить

### Windows:
1. Скачай Python: https://python.org/downloads → установи, отметь "Add to PATH"
2. Открой папку `bot` в проводнике
3. Зажми Shift + правая кнопка мыши → "Открыть окно PowerShell здесь"
4. Введи команды:

```
pip install -r requirements.txt
python bot.py
```

5. Если видишь `✅ Бот запущен` — всё работает!

### Чтобы бот работал 24/7 (бесплатно):

Используй **Railway.app**:
1. Зайди на https://railway.app → войди через GitHub
2. "New Project" → "Deploy from GitHub repo" → выбери свой репо
3. Укажи папку `bot` как корневую
4. В переменных окружения (Variables) добавь:
   - `BOT_TOKEN` = твой токен
5. Нажми Deploy — бот запустится и будет работать постоянно

## ШАГ 5 — Синхронизация заказов с сайтом

Сайт хранит заказы в `localStorage` браузера. Чтобы бот их видел:

**Вариант A (ручной, простой):**
1. Открой сайт → войди в админку
2. Вкладка "Заказы" → кнопка "📤 Экспорт JSON"
3. Скопируй файл `orders.json` в папку `bot/`
4. Бот сразу увидит заказы

**Вариант B (автоматический через GitHub):**
1. Сайт при каждом заказе сохраняет в localStorage
2. Ты экспортируешь → коммитишь в репо → Railway подтягивает автоматически

---

## Как покупатель получает роль

1. Покупатель делает заказ на сайте
2. Ему показывается **ID заказа** (число)
3. Он заходит на твой Discord сервер
4. Пишет боту: `/claim 1234567890`
5. Бот проверяет заказ → выдаёт роль → уведомляет тебя в лог-канал

---

## Команды бота

| Команда | Кто | Что делает |
|---|---|---|
| `/claim <id>` | Покупатель | Получить роль по ID заказа |
| `/orders` | Только ты | Список новых заказов |
| `/done <id>` | Только ты | Отметить заказ выданным |
| `/give_role @user @роль` | Только ты | Выдать роль вручную |
| `/take_role @user @роль` | Только ты | Забрать роль |

---

## Важно про роли

Роль бота в настройках сервера должна быть **ВЫШЕ** всех ролей которые он выдаёт.
Настройки сервера → Роли → перетащи роль бота вверх.
